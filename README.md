# PulseUI Static Frontend

A modern, responsive, static frontend project built with plain HTML, CSS, and JavaScript.

## What's included
- Sticky responsive navbar with mobile toggle
- Hero section with call-to-action buttons
- Feature cards, showcase panels, pricing section, and FAQ
- Clean gradient/glass style inspired by current landing page trends
- Zero-build setup (open `index.html` directly)

## Run locally
```bash
python3 -m http.server 4173
```
Then open `http://localhost:4173`.
